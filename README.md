# 产品营销工作台

产品营销工作台是一个面向产品营销、增长顾问和创业团队的网页版分析工具。它可以基于产品信息生成用户画像、情绪动因、产品机会、外部信号和 AI 深度报告，并支持联网检索、社区舆论、历史记录、后台配置和图片提示词/图片生成工作流。

## 功能

- 登录/注册后进入工作台
- 后台管理 API 配置、用户概览和历史报告
- 产品画像、情绪洞察、机会优先级和外部信号分析
- 全网联索与社区舆论检索
- Markdown 报告渲染、目录定位和重点高亮
- 基于当前产品和联网证据生成图片提示词
- 一键生成多张候选图片，并支持点击放大预览

## 本地运行

```bash
node server.js
```

默认访问：

```text
http://127.0.0.1:5174/
```

## 部署

服务端是原生 Node.js 静态/API 服务，不依赖构建步骤。部署时上传以下文件即可：

- `index.html`
- `styles.css`
- `app.js`
- `server.js`
- `manifest.json`
- `sw.js`
- `assets/auth-face-bg.jpg`

API 地址、令牌、模型名和图片模型配置都保存在浏览器本机存储中，不应写入代码仓库。
